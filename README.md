## キョウガチャ
- ガチャを回すと京都で、今日遊ぶ場所の候補地を排出します。

# 使い方
- 通常ガチャを回す場合は遊ぶ場所の条件を設定し、回すと条件に当てはまる場所の中からランダムで一つ候補地を排出します。
- 闇鍋ガチャの場合は設定を選ぶ必要はありません。
